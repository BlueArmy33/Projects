# C++ Projects

This folder contains C++ projects developed for coursework and personal practice. The projects focus on object oriented design, terminal based user interfaces, and problem solving using C++.

## Skills Demonstrated

- C++ programming
- Object oriented design and class based architecture
- Terminal user interface development using ncurses
- Control flow, input validation, and debugging
- Random number generation and game logic

## Projects Included

### Knucklebones Game
A terminal based dice game implemented in C++ using the ncurses library. The game features two player turn based gameplay, a visual grid displayed in the terminal, score tracking, and win detection.

Technologies used
- C++
- ncurses

Additional C++ projects may be added as development continues.

## How to Run

Each project contains its own source file. To compile and run a project, navigate to its directory and use a C++ compiler.

Example
```bash
g++ knucklebones.cpp -lncurses -o knucklebones
./knucklebones
