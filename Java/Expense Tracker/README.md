# Expense Tracker Java Swing

A simple GUI based expense tracker built with Java Swing. Users can add expenses with an amount and description, view a list of all expenses, calculate the total spent, and clear all saved entries.

This project was created for CMPS 4143 Fall 2025.

## Features

- Add an expense with amount and description
- View all recorded expenses in a numbered list
- Calculate total amount spent
- Clear all expenses with a confirmation prompt
- Credits dialog

## Tech Stack

- Java
- Java Swing
- ArrayList for in memory storage

1. Compile
```bash
