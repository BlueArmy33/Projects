# Java Projects

This folder contains Java projects developed as part of my Computer Science coursework and personal practice. The projects demonstrate core Java programming concepts, graphical user interface development, and problem solving using object oriented design.

## Skills Demonstrated

- Object oriented programming
- Java Swing GUI development
- Event driven programming
- Data handling using collections
- Input validation and error handling

## Projects Included

### Expense Tracker
A GUI based expense tracking application built with Java Swing. The program allows users to add expenses with descriptions, view all recorded expenses, calculate the total amount spent, clear entries, and view program credits.

Technologies used
- Java
- Java Swing
- ArrayList

Additional projects may be added as development continues.
